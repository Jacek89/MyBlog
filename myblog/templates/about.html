About

This website has been created for educational purposes. It can be successfully used as a personal blog and be expanded. During the development of this web application, I studied a lot about how websites work. The main topics I learned:

  • Bootstrap to create basic front-end
  • Flask framework and files structure for such an application
  • CRUD operations with Flask and SQLalchemy
  • SQL models with links
  • PostgreSQL database management, migrations, pgAdmin 4
  • utilize small flask modules to improve application
  • authentication system, handling with user and admin permissions
  • Google authorization
  • the idea of passwords hashing and implementing it to a website
  • sending reset password mails with tokens via JWT
  • using cache to store temporary data with Redis
  • async functions like sending emails, foundations of Celery with Redis
